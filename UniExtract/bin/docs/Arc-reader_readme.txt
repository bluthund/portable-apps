binary of https://github.com/minirop/arc-reader
ethornell.exe helps you extract files from .arc used by BGI
to use:
	- you can drag and drop a .arc file on ethornell.exe *WARNING* it will extract all the files in the same directory as ethornell.exe
	or
	- with cmd.exe, go to where ethornell.exe is and type: ethornell <file>.arc <directory>
	  it will extract all the files is the directory named <directory> (created if not existing)
enjoy
